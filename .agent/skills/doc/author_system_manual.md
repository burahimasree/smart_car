# Author System Manual

The final output: "The Book".

## When to use
- Phase 8 (Book Creation).
- When a major subsystem is verified.

## Step-by-Step Instructions
1. **Gather Artifacts**: Collect all verified reports and SKILL.md files.
2. **Structure**:
   - Front Matter (Version, Date).
   - Introduction (High-level architecture).
   - Chapters (Subsystems).
   - Appendices (API Reference).
3. **Write**:
   - Use clear, professional English.
   - Link to actual files `[code](file://...)`.

## Verification Checklist
- [ ] No broken links.
- [ ] All claims are backed by "Verified" reports.

## Rules & Constraints
